## Adding New Sources

When adding a new source to this index:

1. **Assign a Source ID** — Use format: `PUBLISHER-SHORTNAME` (e.g., `RF-TAE-2026`)
2. **Determine Grade** — A (Government/Official), B (Established CTI Vendor), C (Community/Independent), D (Single Source/Unverified), E (Self-Reported/Marketing)
3. **Save the source file** — PDFs go in this `/sources/` folder. Web articles should be saved as PDF screenshots (like the RF article) or as markdown exports
4. **Add entry to this index** — Include all fields
5. **Update BPH_Master.csv** — Add the Source ID to the `sources` column of any providers mentioned in the new source
6. **File naming convention:** `[Publisher]-[ShortTitle]-[Date].pdf` or `screencapture-[domain]-[path]-[date].pdf`
